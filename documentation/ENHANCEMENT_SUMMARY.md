## 📋 **Quick Start Guide**

### 1. **Environment Setup**
```bash
# Required API keys in .env file
ALPACA_API_KEY_ID=your_key_here
ALPACA_SECRET_KEY=your_secret_here
POLYGON_API_KEY=your_key_here
FINNHUB_API_KEY=your_key_here
```

### 2. **Run Verification**
```bash
node verify.js
```

### 3. **Start Trading**
```bash
node BitFlow.js BTC/USD
```
